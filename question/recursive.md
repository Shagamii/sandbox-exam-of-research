## 問題

2 ~ 20 番目(n)のフィボナッチ数をそれぞれ n-1 番目の数字と掛け合わした数を表せ.
また, 掛け合わせの関数は再帰関数によってのみ実装可能とする.

n の条件は以下とする.

```
2 < 20
```

### フィボナッチ数列

```
Cn+1 = Cn + Cn-1
```
